## How to Use This Changelog

### Types of Changes

- **Added** - New features
- **Changed** - Changes in existing functionality
- **Deprecated** - Soon-to-be removed features
- **Removed** - Removed features
- **Fixed** - Bug fixes
- **Security** - Security vulnerability fixes

### Version Format

Use [Semantic Versioning](https://semver.org/):
- **MAJOR** version for incompatible API changes
- **MINOR** version for backwards-compatible functionality additions
- **PATCH** version for backwards-compatible bug fixes

### Example Entry

```markdown
## [1.2.0] - 2026-03-15

### Added
- Export case data to PDF functionality (#123)
- Advanced search filters for evidence (#124)
- Multi-file upload with drag-and-drop (#125)

### Changed
- Improved timeline rendering performance (#126)
- Updated network graph layout algorithm (#127)

### Fixed
- Fixed authentication token refresh issue (#128)
- Resolved pagination bug in case list (#129)
- Corrected date formatting in timeline view (#130)

### Security
- Updated axios to 1.6.0 to patch CVE-2023-XXXXX (#131)
```
